# High-sensitive flexible dual-modal flow sensing array for accurate vortex source localization

This repository contains data, codes, and models for vortex source localization using advanced dual-modal flow sensing technology. The project demonstrates high-sensitivity flow detection capabilities across multiple experimental scenarios.

## Cylinder localization

Contains experimental data, algorithms, and models for localizing vortex sources generated by cylinders in flow environments. This section includes:

- Experimental datasets from cylinder flow experiments
- Signal processing algorithms for vortex detection
- Machine learning models trained on cylinder localization data
- Performance metrics and validation results

## Fixed robotic fish localization

Focuses on localizing vortex sources from stationary robotic fish configurations. This section includes:

- Data collection protocols from fixed robotic fish experiments
- Feature extraction and preprocessing techniques
- Localization algorithms optimized for static fish positioning
- Calibration and validation procedures
- Experimental results and performance analysis

## Swimming robotic fish localization

Addresses the more challenging problem of localizing vortex sources from actively swimming robotic fish. This section includes:

- Dynamic flow data from swimming robotic fish experiments
- Real-time signal processing methods
- Adaptive algorithms for moving vortex sources
- Tracking and localization frameworks
- Comparative analysis with fixed fish scenarios

---

**Key Features:**
- Dual-modal sensing for enhanced sensitivity
- Flexible sensor array design
- Accurate vortex source position estimation
- Comprehensive experimental validation

For more information about the methodology and results, please refer to the individual folders and accompanying documentation.
